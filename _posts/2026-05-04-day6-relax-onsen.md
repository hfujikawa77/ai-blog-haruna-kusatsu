## 🏛️ 榛名山エリア：伊香保石段街＆昭和レトロ体験！

### 伊香保温泉 石段街＆建築散歩

[伊香保温泉](https://harunavi.jp/)の象徴、[石段街](https://harunavi.jp/)は温泉の源泉に沿って365段の石段が続く参道。江戸時代から続く温泉街の風情がそのまま残っていて、歩くだけでタイムスリップした気分になれる。

**石段街の基本情報**

| 項目 | 内容 |
|------|------|
| 段数 | 365段（1年365日を意味するとも言われる） |
| 全長 | 約250m |
| 両側 | みやげ店・射的・旅館・カフェが立ち並ぶ |
| 見どころ | 石段の一番上にある[伊香保神社](https://harunavi.jp/)、古い温泉建築 |

**石段街で楽しめること**

| 体験 | 内容 |
|------|------|
| 射的 | 木製の的を狙って撃つ昔ながらのゲーム。コルクの弾が飛ぶ！ |
| 縁日屋台 | 輪投げ・スーパーボールすくいなど昭和の縁日 |
| 温泉まんじゅう食べ歩き | 石段を登りながら食べるのがここのスタイル |
| 温泉たまご | 源泉で自然に茹でた温泉卵。黄身がトロトロ！ |
| 足湯 | 石段途中にある無料の足湯でひと休み |

> 💡 石段は途中でも休めるベンチが各所にある。ゆっくり両脇の店を見ながら登るのがおすすめ。頂上の[伊香保神社](https://harunavi.jp/)でお参りすれば旅の安全を祈れる！

---

### 射的・縁日・昭和レトロ体験

伊香保の石段街にある射的屋さんは、昭和から変わらないスタイル。コルクの弾を使った木製ライフルで、棚の上の景品を狙う。

**射的のコツ**

1. 銃の構え方はスタッフが教えてくれる（初心者OK）
2. 目線と銃口をまっすぐ揃えるのがポイント
3. 景品の角を狙うと倒れやすい
4. 弾は5発〜10発セット購入が基本

> ✅ 的を倒せなくても弾が当たれば景品がもらえる場合も。まずは挑戦してみよう！

**伊香保らしい建築散歩**

石段の両脇には大正・昭和初期の木造建築が残っている。窓の格子・屋根の形・石造りの基礎など、今ではなかなか見られない建築様式を観察するだけで楽しい。

## ♨️ 草津エリア：湯もみショー＆草津ホテル1913で温泉文化に浸ろう！

### 湯もみショー（草津の伝統文化）

草津温泉を代表する伝統文化「湯もみ」。高温（約50〜60℃）の源泉を長い板でかき混ぜて温度を下げ、入浴できる温度に調整する昔ながらの技法。[熱乃湯（ねつのゆ）](https://kusatsu-kanko.com/experience/netsunoyu/)でその実演ショーを見ることができる。

**湯もみショー基本情報**

| 項目 | 内容 |
|------|------|
| 場所 | [熱乃湯](https://kusatsu-kanko.com/experience/netsunoyu/)（湯畑目の前） |
| 上演時間 | 約30分 |
| 料金 | 大人700円・小人400円 |
| 公演回数 | 1日6回（時間は公式サイト確認） |
| 内容 | 湯もみの実演＋草津節の踊り |

> 💡 ショー後半では**観客が湯もみ体験に参加できる**こともある！あの長い板を握って湯をかき混ぜる感覚はここでしか味わえない。

### 草津ホテル1913の施設・温泉ガイド

今夜の宿泊先、[草津ホテル1913](https://www.kusatsuhotel.com/)は大正2年（1913年）創業の老舗旅館。[西の河原公園](https://onsen-kusatsu.com/sainokawara/)の入口という絶好の立地で、全40室の落ち着いた宿。

**施設ハイライト**

| 施設 | 特徴 |
|------|------|
| 貸切家族風呂 | プライベートな温泉を家族だけで独占。時間貸し制 |
| 大浴場 | 草津の源泉を使用。疲れた体をしっかり温める |
| お食事処 | 群馬の食材を使った懐石・会席料理 |
| 立地 | [西の河原公園](https://onsen-kusatsu.com/sainokawara/)まで徒歩すぐ・湯畑まで徒歩約5分 |

> ✅ 貸切家族風呂は事前予約制。チェックイン時に空き状況を確認しよう。3人家族でも使いやすいサイズの浴室が多い。

---

### 夜の湯畑ライトアップ

夜の[湯畑](https://www.kusatsu-onsen.ne.jp/)はライトアップされて昼間とはまったく違う幻想的な雰囲気に。白い湯煙と硫黄の香りの中、青白い光に照らされた湯畑は写真映えも最高。

**夜の湯畑散策のポイント**

- ライトアップは日没後（19時〜21時頃）がベスト
- 湯畑を囲む遊歩道を一周：所要約15〜20分
- [西の河原公園](https://onsen-kusatsu.com/sainokawara/)まで足を延ばすと夜の足湯も楽しめる
- 夜は気温が下がるので上着を忘れずに

> 💡 湯畑のライトアップは青・白・オレンジのグラデーション。スマホのカメラで夜景モードを使えば美しく撮れる！

## 🧠 今日のクイズ

**Q. 草津温泉の「湯もみ」はただかき混ぜるだけじゃない。温泉成分を変えずに温度を下げる科学的な理由とは？**

<details>
<summary>答えを見る</summary>

**答え：水を加えて薄めると薬効成分が変わってしまうから！**

草津の源泉は50〜60℃という高温。水を加えて薄めてしまうと薬効成分が変わってしまうため、温度を均一にかき混ぜることで入浴できる温度（約42℃）まで下げる。長い板を使うのは表面積を最大化して熱を逃がしやすくするから。昔の人の知恵がすごい！

</details>
