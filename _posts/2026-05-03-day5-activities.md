GW5日目は体を動かしまくる日！今夜は[サンコーハルナパーク](https://harunapark.jp/)でバンガロー泊＆BBQという最高のフィナーレが待っている。ロープウェイ・ヤマメ釣り・パン作り・草津のアクティビティ……全部やりきろう！

## 🏕️ 今夜の宿：サンコーハルナパーク バンガロー泊＆BBQ

[サンコーハルナパーク](https://harunapark.jp/)は榛名山南麓に広がるアウトドアリゾート。今夜はここでバンガロー泊＋屋根付きBBQを楽しもう！

| 施設 | 内容 |
|------|------|
| バンガロー | エアコン・ベッド・トイレ付き。自然の中でも快適に過ごせる |
| 屋根付きBBQ | 雨天でもOKな屋根付き炉。食材の持ち込み・現地購入どちらも可 |
| ヤマメ釣り | 場内の沢で体験可能。竿・エサは貸出あり |
| パン作り体験 | 事前予約制。焼きたてパンをBBQの前菜に！ |
| 沢遊び | 場内を流れる清流で水遊び可能 |

> 💡 パン作りは人気体験のため、予約は早めに。BBQとの組み合わせで満足度が倍増！

## 🚡 榛名山エリア：ロープウェイ＆アクティビティ全開！

### 榛名富士ロープウェイ 乗車ガイド

[榛名富士ロープウェイ](https://www.tanigawadake-rw.com/haruna/)は、榛名湖畔の乗り場から榛名富士山頂（1,391m）まで約3分で登れる空中散歩。360度の眺望と山頂の榛名富士山神社が待っている！

**乗車基本情報**

| 項目 | 内容 |
|------|------|
| 運行区間 | 榛名湖畔→榛名富士山頂 |
| 所要時間 | 片道約3分 |
| 料金 | 大人往復650円・小人往復330円 |
| 運行時間 | 9:00〜17:00（季節により変動） |
| 駐車場 | 無料・約50台 |

> ✅ 山頂からは天気がよければ関東平野・赤城山・谷川岳が見渡せる絶景！写真を撮る余裕を持って乗ろう。

**山頂でできること**

- [榛名富士山神社](https://www.tanigawadake-rw.com/haruna/)の参拝とご朱印GET
- 360度パノラマ展望
- 山頂カフェでひと休み（コーヒー・甘酒など）

### キャンプ場内の沢遊び・ヤマメ釣り・パン作り体験

[サンコーハルナパーク](https://harunapark.jp/)内では、アウトドア体験がぎっしり揃っている。BBQの前後にしっかり楽しもう！

**ヤマメ釣り体験**

| 項目 | 内容 |
|------|------|
| 場所 | キャンプ場内を流れる沢 |
| 料金 | 1匹あたり料金制（目安：500〜700円/匹） |
| 道具 | 竿・エサは全部貸し出し |
| 釣った魚 | その場で塩焼きにしてBBQと一緒に食べられる |

> 💡 ヤマメは「渓流の女王」と呼ばれるきれいな魚。釣り未経験でも1時間あれば1匹は釣れるはず。粘り強く待つのが釣りの醍醐味！

**パン作り体験**

| 項目 | 内容 |
|------|------|
| 所要時間 | 約60〜90分 |
| 内容 | 生地をこねて、成形して、焼くまでの全工程 |
| 予約 | 事前予約制（場内の受付または公式サイトで確認） |

**沢遊び**

場内を流れる清流は水温が低く（夏でも15〜18℃）、ジャブジャブ水遊びができる。サンダル・着替え必須！

## 🎿 草津エリア：夏の草津でアクティビティ体験！

### 草津スキー場の夏シーズン施設

[草津国際スキー場](https://www.kusatsu-kokusai.com/)は冬のスキーだけじゃない！夏シーズン（GW〜秋）はゲレンデを利用した特別アクティビティが楽しめる。

**夏のアクティビティ**

| 体験 | 内容 | 特徴 |
|------|------|------|
| マウンテンバイク | ゲレンデの斜面を自転車で駆け下りる | リフト乗車で山頂まで上がれる |
| ゴーカート | 特設コースを疾走 | 免許不要・子どもでも運転可 |
| ハイキング | 高原の眺望を楽しみながら散策 | 初心者向けコースあり |
| リフト乗車のみ | ゲレンデ上部からの眺望を楽しむ | 所要約20分 |

> ⚠️ 草津国際スキー場の夏営業は年によって変動があるため、[公式サイト](https://www.kusatsu-kokusai.com/)で最新情報を確認してから向かおう。

**アクセス**

| 項目 | 内容 |
|------|------|
| 草津温泉からの距離 | 車で約10分 |
| 駐車場 | 無料・広大なスキー場駐車場 |

### ガラス工芸・陶芸など体験コーナー

草津温泉の[湯畑](https://www.kusatsu-onsen.ne.jp/)周辺や[西の河原公園](https://onsen-kusatsu.com/sainokawara/)エリアにはクラフト体験施設がある。

**体験メニュー**

| 体験 | 内容 | 所要時間 |
|------|------|---------|
| ガラス工芸（吹きガラス） | 熱したガラスを吹いてオリジナルのグラス・小物を作成 | 約60〜90分 |
| 陶芸体験 | 電動ろくろまたは手びねりで器を制作 | 約60〜90分 |
| キャンドル作り | 硫黄カラーキャンドル制作 | 約30〜60分 |

> 💡 吹きガラス体験は草津でも数か所で行われている。旅の記念になるオリジナル作品を持ち帰れる！

## 🧠 今日のクイズ

**Q. ヤマメが「渓流の女王」と呼ばれる理由はなんでしょう？**

<details>
<summary>答えを見る</summary>

**答え：きれいな水にしか住めない「水質指標生物」だから！**

ヤマメはきれいで冷たい水（水温20℃以下）にしか生息できない。そのため「ヤマメがいる川＝きれいな川」の証明になる。体側の美しいパーマーク（楕円形の模様）も「女王」の名前の由来。食べてもおいしい川魚のエース！
